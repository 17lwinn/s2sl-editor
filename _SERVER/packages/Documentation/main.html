FAQ

Before we begin, take a look at the FAQ.

What is auroraOS?

Think Microsoft Windows or macOS, but in your browser.

I found a bug. Where do I report it?

https://github.com/soupy-developer/auroraOS/issues.

How do I install auroraOS on my own server?

Currently, there isn't an automated installer. However, you can install it manually by cloning the git repository and running _SERVER/server.js. Then, you can visit localhost:[the port, default is 3000] or configure it.
