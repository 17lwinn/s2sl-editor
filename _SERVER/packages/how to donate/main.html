How To Donate!

welcome to Scripting To Save Lives!

this editor will allow you to donate code to us directly!

HOW TO DONATE:

    1. go to the start menu and click 'notepad'
    2. type your code and save to the topic folder it is used for (eg- website/index.html)
    3. go to the file manager and go to the place your file was saved to, click to open it
you can only view html files dynamically, if you want you may link seperate files- they must have some
kind of sign that you own them

WARNING! once you save your file it cannot be renamed,deleted or moved!
